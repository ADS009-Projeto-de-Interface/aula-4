Frontend

Frontend refere-se à parte de um sistema ou aplicação que os usuários interagem diretamente. É a interface visível e interativa através da qual os usuários interagem com um site, aplicativo ou software. O desenvolvimento frontend envolve a criação e implementação de elementos visuais, como layouts, botões, menus e outros componentes gráficos, bem como a programação que permite a interatividade do usuário. Os desenvolvedores frontend utilizam linguagens de marcação, como HTML, estilização com CSS e programação com JavaScript para criar uma experiência de usuário atraente e funcional. Essa camada do desenvolvimento é crucial para proporcionar uma interface intuitiva e responsiva, garantindo uma interação suave e eficiente entre os usuários e a aplicação.

HTML5

HTML5 (Hypertext Markup Language 5) é a quinta versão do padrão HTML, que é a linguagem de marcação fundamental para a construção de páginas web. Lançado em outubro de 2014, o HTML5 trouxe consigo uma série de recursos e melhorias em relação às versões anteriores. Entre suas principais características, estão novas tags que facilitam a estruturação do conteúdo, suporte nativo para áudio e vídeo, elementos de formulário mais avançados, APIs para interação com dispositivos e armazenamento local, além de oferecer melhor suporte para gráficos e animações. O HTML5 desempenhou um papel significativo na promoção de uma web mais rica em recursos, acessível e compatível com uma variedade de dispositivos e navegadores. Ele continua a ser amplamente adotado na criação de páginas web modernas e responsivas.

CSS3

CSS3 (Cascading Style Sheets 3)é a terceira versão da linguagem de estilo utilizada para controlar a apresentação visual de documentos HTML e XML. CSS descreve como os elementos HTML devem ser exibidos na tela, permitindo o controle de layout, cores, fontes, espaçamento e outros aspectos visuais de uma página web. CSS3, introduzido em 1999, trouxe uma série de novos recursos e aprimoramentos em relação às versões anteriores. Algumas das características notáveis incluem seletores avançados, animações, sombras, gradientes, bordas arredondadas, transições e transformações. Esses recursos permitem aos desenvolvedores criar designs mais complexos e interativos sem depender tanto de imagens ou scripts adicionais. CSS3 é amplamente utilizado na criação de layouts modernos e responsivos para páginas web.

Javascript 6+

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos. Originalmente desenvolvida para ser executada nos navegadores web, JavaScript tornou-se uma parte fundamental do desenvolvimento web moderno, permitindo interatividade dinâmica e manipulação de conteúdo em páginas web. Essa linguagem é utilizada para criar elementos interativos, validar formulários, criar animações, fazer requisições assíncronas (AJAX) para atualizações de dados em segundo plano, entre outras funcionalidades.
